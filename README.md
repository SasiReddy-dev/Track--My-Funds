📱 Track My Funds – Django Finance Tracker
🔹 Project Description
Track My Funds is a web-based personal finance management application developed using Python and Django. The application allows users to record their income and expenses, manage financial data, and visualize spending through charts in a simple and user-friendly interface.
🚀 Features
User Registration and Login (Authentication)
Add Income Details
Add Expense Details
Edit and Delete Expenses
Category-based Expense Management
Expense Chart Visualization
Secure User Authentication
Clean and Responsive UI

🛠️ Technologies Used
Backend: Python, Django
Frontend: HTML, CSS, JavaScript
Database: SQLite
Version Control: Git & GitHub

⚙️ How to Run This Project
Step 1: Clone the repository
git clone https://github.com/SasiReddy-dev/Track--My-Funds.git

Step 2: Navigate to the project folder
cd Track--My-Funds

Step 3: Create a virtual environment
venv\Scripts\activate

Step 5: Install dependencies
pip install -r requirements.txt

Step 6: Run the server
python manage.py runserver

Open in browser:
http://127.0.0.1:8000/
